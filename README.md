# skperry99.github.io

Personal portfolio site for Sarah Perry, built with GitHub Pages (Jekyll) and the Cayman theme.

## Local preview (optional)

If you want to run it locally:

1. Install Ruby + Bundler
2. From the repo root:

```bash
bundle install
bundle exec jekyll serve
```

Then open the local URL Jekyll prints (usually http://127.0.0.1:4000).

## Customize

- Site settings: `_config.yml`
- Layout: `_layouts/default.html`
- Pages: `index.md`, `about.md`, `projects.md`, `resume.md`, `contact.md`
- Custom styles: `assets/css/custom.css`

## Notes

This repo ignores local-only files (IDE settings, lockfiles, build output) via `.gitignore`.
